# Quick Setup Guide

## Option 1: Using Batch Files (Recommended for Windows)

1. **Install Dependencies**:
   - Double-click `install.bat` to install all dependencies
   - Wait for the installation to complete

2. **Start Development Server**:
   - Double-click `start.bat` to start the development server
   - Open http://localhost:3000 in your browser

## Option 2: Using Command Line

If you have PowerShell execution policies enabled:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
